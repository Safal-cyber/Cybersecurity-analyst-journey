# SOC Incident Detection & Response Simulation

## Objective
Simulate real-world cyber attack and perform detection, analysis, and response.

## Tools Used
- Linux (Ubuntu)
- Splunk
- tcpdump
- Wireshark

## Attack Simulation
- Brute-force login attempts
- Suspicious process execution
- Outbound network communication
- Persistence via cron

## Detection Steps
- Log analysis (auth.log)
- Splunk investigation
- Process monitoring
- Network analysis

## Response Actions
- Killed malicious process
- Blocked attacker IP
- Removed persistence

## Outcome
Successfully detected and mitigated simulated attack.

## Skills Demonstrated
- SOC analysis
- Incident response
- Log correlation
- Threat detection
